Support

If you need help with iLand, you can contact support using the details below.

Contact

Email: ss@codeiland.com
Website: iland.codeiland.com/support

Helpful Information

When contacting support, please include:

  • Your device model
  • Your iOS version
  • Your app version
  • A short description of the issue
  • Screenshots, if relevant

Common Topics

  • Document scan issues
  • Passport NFC reading issues
  • Permission issues for camera, NFC, or location
  • Profile and identity onboarding questions
  • App icon customization
